# Royal for Pipes & Building Materials

واجهة عربية RTL لشركة Royal للمواسير وأدوات السباكة والكهرباء ومواد البناء.

## ما الذي يتضمنه المشروع

- صفحة رئيسية بهيرو متحرك وعرض أقسام رئيسية وأحدث المنتجات والوكلاء.
- كتالوج منتجات مع تصفية حسب القسم والتصنيف.
- صفحة للوكلاء والموزعين حسب المحافظات اليمنية.
- صفحة عن الشركة مع صور ومساحة للفيديو التعريفي.
- صفحة اتصل بنا مع نموذج حفظ الرسائل وخريطة ومعلومات التواصل.
- لوحة تحكم للمدير مع تسجيل دخول وإدارة المنتجات والأقسام والوكلاء والعروض والرسائل والإعدادات.
- تخزين محلي Local Storage لحفظ التعديلات داخل المتصفح.
- ملف قاعدة بيانات MySQL مرفق باسم `database.sql` كبداية للربط مع PHP/MySQL.

## بيانات الدخول الافتراضية للمدير

- اسم المستخدم: `admin`
- كلمة المرور: `royal123`

## التشغيل محلياً

1. ثبت الحزم:
   ```bash
   npm install
   ```
2. شغّل المشروع:
   ```bash
   npm run dev
   ```
3. افتح الرابط المحلي الذي يظهر في الطرفية.

## البناء للإنتاج

```bash
npm run build
```

## قاعدة البيانات

- الملف `database.sql` يحتوي على الجداول الأساسية: الأقسام، المنتجات، الوكلاء، العروض، الرسائل، والإعدادات.
- يمكنك استيراده في MySQL ثم استبدال طبقة Local Storage بطبقة PHP لاحقاً.

## ملاحظات مهمة

- الصور المستخدمة في الواجهة عبارة عن صور حقيقية من Pexels.
- التصميم جاهز كنقطة انطلاق قوية للاستضافة الحقيقية بعد ربطه بواجهة PHP/MySQL.
- إذا أردت نسخة PHP كاملة لاحقاً، يمكن تحويل نفس البنية إلى Laravel أو PHP خام بسهولة.
